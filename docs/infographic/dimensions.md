# 信息图五维度

本文档维护信息图的五个维度及其可选值，供 Skill 与生成器一致引用。可后续增删选项。

## 1. 比例（画布宽高比）

| 选项   | 说明       |
|--------|------------|
| 16:9   | 横屏宽屏   |
| 4:3    | 横屏标准   |
| 1:1    | 正方形     |
| 3:4    | 竖屏标准   |
| 9:16   | 竖屏全屏   |

## 2. 风格（视觉风格）

| 选项   | 说明                     |
|--------|--------------------------|
| 扁平   | flat，无立体阴影         |
| 手绘   | hand-drawn，线条/涂鸦感  |
| 渐变   | gradient，多色渐变       |
| 商务   | business，正式、图表感   |
| 极简   | minimal，留白、少元素     |
| 插画   | illustration，插画化     |

## 3. 布局（版式结构）

| 选项     | 说明           |
|----------|----------------|
| 单栏     | single-column  |
| 多栏     | multi-column   |
| 网格     | grid           |
| 时间轴   | timeline       |
| 中心放射 | radial/center  |
| 左右分栏 | two-column     |

## 4. 语言（文案/界面语言）

| 选项   | 说明     |
|--------|----------|
| zh-CN  | 简体中文 |
| en     | 英文     |

（可按需扩展如 zh-TW、ja 等。）

## 5. 配色（主色/色板）

| 预设名   | 说明           |
|----------|----------------|
| ocean    | 蓝绿色系       |
| warm     | 暖色（橙/红/黄）|
| mono     | 单色/灰度      |

也可用色值说明（如 HEX 或主色 + 辅色）在具体需求中指定。
